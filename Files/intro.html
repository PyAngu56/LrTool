LightRoom Tools

Lightroom Tools is an experiment to play with the Lightroom database.
The Python server analyzes the database and uses OpenCV to recognize faces and do crop poposal.
A Web browser is used to present the results.

WORK IN PROGRESS !
USE THIS AT YOUR OWN RISK. SINCE THE DATABASE IS CHANGED, ALWAYS WORK WITH A BACKUP !
